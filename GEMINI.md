# Project: EstacionamientoAutos

## Project Overview

This is a C# Windows Forms application for a parking lot management system, named "EstacionamientoAutos". The project is built using .NET 8 and follows the standard structure for a WinForms application. The main entry point is `Program.cs`, which launches the main form, `Form1`.

## Building and Running

To build and run this project, you can use the following standard `.NET CLI` commands:

*   **Build the project:**
    ```bash
    dotnet build
    ```
*   **Run the application:**
    ```bash
    dotnet run
    ```

## Development Conventions

The project follows the standard C# and Windows Forms development conventions.

*   **UI and Logic Separation:** The UI is defined in the `Form1.Designer.cs` file, and the corresponding logic is in the `Form1.cs` file.
*   **Entry Point:** The application's entry point is the `Main` method in the `Program.cs` file.
*   **Dependencies:** Project dependencies and properties are managed in the `EstacionamientoAutos.csproj` file.
